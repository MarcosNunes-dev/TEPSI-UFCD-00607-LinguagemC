# 💻 Formação CET TPSI - Lógica e Linguagem C

Este repositório contém a minha progressão prática no módulo de Programação em C. Os exercícios foram numerados e organizados para demonstrar a evolução desde os fundamentos até estruturas de dados complexas.

## 📑 Índice de Exercícios
Os ficheiros seguem uma sequência lógica de aprendizagem:

* **01-03:** Fundamentos, tipos de dados e cálculos aritméticos.
* **04-06:** Estruturas de controlo de fluxo (`switch`, `if-else`) e ciclos (`for`).
* **07-09:** Lógica de problemas (áreas, calculadoras e tabuadas com validação).
* **10-12:** Manipulação de Arrays e Strings (vetores e ordenação).
* **13-14:** Gestão de memória: Ponteiros e Passagem por Referência.
* **15-17:** Estruturas de dados avançadas: Memória Dinâmica, Structs e Typedef.

## 🛠️ Tecnologias e Conceitos
* **Linguagem:** C
* **Compilador:** MSVC (Visual Studio) / GCC
* **Conceitos:** Alocação de memória, ponteiros, lógica condicional e modularização.
